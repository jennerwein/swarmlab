conn-check
=========
Überprüft, ob die angegebenen hosts erreichbar sind.

Requirements
------------
Im hosts-File ist jeder IP-Adresse als Variable ein hostname zugeordnet.
Beispiel:
192.168.178.93 hostname=mgmt-serv.93

Role Variables
--------------
keine

Dependencies
------------
Zuordnung IP-Adresse --> Hostname in hosts
